# Mi script de Utilidad - Crear Proyecto Web
#### Este script nace de la necesidad para crear facilmente un proyecto web Front-End.
Habitualmente me encuentro creando continuamente la estructura de directorio para un proyecto web por mi amor a la ense√±anza. No obstante me resulta tedioso rearmar continuamente toda la estructura de directorio; podria tener armado una carpeta y renombrarla pero me interesa poder tener un scripto que lo haga cada vez que lo requiera.

El script, al ejecutarse, solicita el nombre del proyecto y construye la siguiente estructura de directorios.
```
    NameWebProyect
        |
        +---css
        +---img
        +---js
        +---pages
        +---recursos
        +---index.html
```
